Introdução ao HTML

HTML (Hyper Text Markup Language) é uma linguagem de marcação de hipertexto utilizadas apra a criação de paginas na Web. Essa é a linguagem que o seu navegador (Browser) decodifica para exibir as páginas. (FERREIRA & EIS, ONLINE, p.7)

Breve Histórico da Evolução do HTML

Desde a invenção da Web em 1992 por Tim Berners-Lee, a HTML evoluiu para cada versão lançada, sendo a versão mais atual a HTML5.
